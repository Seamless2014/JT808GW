# JT808GW
本网关当前并非全功能网关，主要实现与RTVS对接演示。

当前仅实现了常用协议层面接入和网关公共逻辑(如：指令、分包、校验等)，所有消息收到后均只应答协议内容，不做与RTVS无关的逻辑处理，且并未保存数据，如需用于生产，请自行实现。

注：当前不支持808-2019
